Conformance Test Suite - GML in JPEG 2000

This test suites checks JPEG 2000 image data for conformance against OGC GML in JPEG 2000 (GMLJP2) Encoding Standard Part 1 OGC 08-085r4. The nominal conformance target is a software component (an application or library) that can produce JPEG 2000 codestreams containing supplementary geospatial metadata.
